Instructions:

Explain how to load data from CSV files and perform basic data manipulation using Pandas.

Data Loading from CSV

Pandas provides the read_csv() function to easily load data from CSV files into a DataFrame.

import pandas as pd

# Load data from a CSV file
df = pd.read_csv('data.csv')

# Print the first 5 rows of the DataFrame
print(df.head())

Basic Data Manipulation

Here are some common data manipulation tasks:

  • Selecting Columns:
    • # Select a single column
    column1 = df['column_name']

    # Select multiple columns
    subset = df[['column1', 'column2']]
  • Filtering Rows:
    • # Filter rows based on a condition
    filtered_df = df[df['column_name'] > 10]
  • Adding New Columns:
    • # Add a new column
    df['new_column'] = df['column1'] + df['column2']
  • Dropping Columns:
    • # Drop a column
    df = df.drop('column_name', axis=1)  # axis=1 for columns
