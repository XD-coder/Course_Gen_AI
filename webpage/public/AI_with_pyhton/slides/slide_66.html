Instructions: Exercise - Apply one-hot encoding to the categorical features

Consider the following dataframe with categorical features:

  
    import pandas as pd

    data = {'Color': ['Red', 'Green', 'Blue', 'Red', 'Green'],
            'Size': ['Small', 'Medium', 'Large', 'Small', 'Medium']}

    df = pd.DataFrame(data)

Use pandas to apply one-hot encoding to the 'Color' and 'Size' columns.

Question 1: After one-hot encoding, how many columns will the dataframe have?





Correct Answer: 6 (Color: Red, Green, Blue; Size: Small, Medium, Large)

Question 2: What would be the values in the "Color_Green" column for the first row?



Correct Answer: 0 (The first row has "Color:Red")

Question 3: What is the code snippet to do one hot encoding in pandas?




Correct Answer: pd.get_dummies(df)
