Slide 49: Exercise - Virtual Environment Setup

Objective: Learn to create and manage virtual environments for Python projects, specifically installing NumPy and Pandas.

Instructions:

  1. Create a Virtual Environment: Use the command python -m venv myenv (or python3 -m venv myenv if python defaults to Python 2) to create a virtual environment named "myenv".
  2. Activate the Virtual Environment:
    • Windows: myenv\Scripts\activate
    • macOS/Linux: source myenv/bin/activate
  3. Install NumPy: Run pip install numpy inside the activated virtual environment.
  4. Install Pandas: Run pip install pandas inside the activated virtual environment.
  5. Verify Installation: Open a Python interpreter within the virtual environment and import both NumPy and Pandas to confirm successful installation.
