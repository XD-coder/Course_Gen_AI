Slide 3: Installing Python and pip on Windows

Instructions:

  1. Download Python: Go to the official Python website: https://www.python.org/downloads/windows/ and download the latest stable version of Python for Windows.

    (Replace placeholder_download_python.png with a screenshot of the Python download page)

  2. Run the Installer: Locate the downloaded executable file (e.g., python-3.x.x-amd64.exe) and double-click to run it.

    (Replace placeholder_run_installer.png with a screenshot of running the Python installer)

  3. Important: Check "Add Python x.x to PATH": In the first window of the installer, ensure you check the box that says "Add Python x.x to PATH". This is crucial for making Python and pip accessible from the command line. You can also select "Use admin privileges when installing py.exe" if desired. Then, you can choose "Install Now" for a standard installation or "Customize installation" to modify the installation options. We will continue with "Install Now" in this example.

    (Replace placeholder_add_to_path.png with a screenshot of the "Add Python to PATH" checkbox)

  4. Wait for Installation: Let the installer complete the installation process.

    (Replace placeholder_installation_progress.png with a screenshot of the Python installation progress)

  5. Verify Installation: Open a command prompt (cmd) or PowerShell window. Type python --version and press Enter. If Python is installed correctly, it will display the Python version.

    (Replace placeholder_verify_python.png with a screenshot of the command prompt showing the Python version)

  6. Verify pip Installation: In the same command prompt window, type pip --version and press Enter. If pip is installed correctly, it will display the pip version.

    (Replace placeholder_verify_pip.png with a screenshot of the command prompt showing the pip version)
