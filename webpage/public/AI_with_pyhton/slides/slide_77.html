Quiz 3: Linear Regression Concepts and Implementation

Instructions: Please answer the following questions about linear regression.

Question 1: What is the primary goal of linear regression?




Answer: a. To predict a continuous target variable based on one or more independent variables.

Question 2: Which of the following methods is commonly used to find the best-fit line in linear regression?




Answer: a. Gradient Descent

Question 3: What does the 'slope' represent in a simple linear regression model?




Answer: b. The change in the dependent variable for a one-unit change in the independent variable.

Question 4: Which of the following Python libraries is most commonly used for implementing linear regression?




Answer: c. Scikit-learn

Question 5: What is R-squared in linear regression?




Answer: c. A measure of the goodness of fit of the model to the data.
