Slide 34: NumPy: Array Attributes and Basic Operations

Array Attributes

  • Shape: The dimensions of the array (e.g., (3, 4) for a 3x4 matrix).
  • Dtype: The data type of the array elements (e.g., int64, float64).
  • Size: The total number of elements in the array.
import numpy as np
arr = np.array([[1, 2, 3], [4, 5, 6]])
print("Shape:", arr.shape) # Output: (2, 3)
print("Dtype:", arr.dtype) # Output: int64 (or similar)
print("Size:", arr.size) # Output: 6

Basic Operations

  • Arithmetic Operations: Element-wise addition, subtraction, multiplication, division.
  • Slicing: Accessing parts of an array using indices.
  • Reshaping: Changing the shape of an array without changing its data.
import numpy as np
arr1 = np.array([[1, 2], [3, 4]])
arr2 = np.array([[5, 6], [7, 8]])
addition = arr1 + arr2
print("Addition:\n", addition)
