Slide 20: Functions in Python

Defining Functions

In Python, functions are defined using the def keyword followed by the function name, parentheses (), and a colon :.


  def function_name(arguments):
      # Function body
      return value

Arguments

Functions can accept different types of arguments:

  • Positional Arguments: Passed in the order they are defined.
  • Keyword Arguments: Passed with the argument name (e.g., name="Alice").
  • Default Arguments: Arguments with a default value (e.g., def greet(name="Guest")).
  • *args: Allows passing a variable number of non-keyword arguments.
  • **kwargs: Allows passing a variable number of keyword arguments.

Return Values

Functions can return a value using the return statement. If no return statement is present, the function returns None.


  def add(a, b):
      return a + b

  result = add(5, 3) # result will be 8

Instructions

Here's how to define functions with different argument types and return values:

  1. Use def to start the function definition.
  2. Specify argument names within the parentheses.
  3. Use return to send back a value.
