Slide 33: NumPy: Introduction to Arrays (ndarray)

What is NumPy?

NumPy (Numerical Python) is a fundamental package for numerical computation in Python. It provides:

  • A powerful N-dimensional array object (ndarray)
  • Sophisticated functions for array operations
  • Tools for integrating C/C++ and Fortran code
  • Useful linear algebra, Fourier transform, and random number capabilities

Why use NumPy arrays?

  • Efficiency: NumPy arrays are more efficient than Python lists for numerical operations, especially for large datasets.
  • Functionality: NumPy provides a vast collection of functions for array manipulation, mathematical operations, and more.
  • Convenience: Easier syntax for complex operations on arrays.

Creating NumPy Arrays

The core of NumPy is the ndarray object. Here's how to create one:

1. From a Python List


import numpy as np

# Create a 1D array
my_list = [1, 2, 3, 4, 5]
my_array = np.array(my_list)
print(my_array)  # Output: [1 2 3 4 5]

# Create a 2D array
my_list_of_lists = [[1, 2, 3], [4, 5, 6]]
my_array_2d = np.array(my_list_of_lists)
print(my_array_2d)
# Output:
# [[1 2 3]
#  [4 5 6]]

2. Using NumPy Functions


import numpy as np

# Create an array of zeros
zeros_array = np.zeros((2, 3))  # 2 rows, 3 columns
print(zeros_array)
# Output:
# [[0. 0. 0.]
#  [0. 0. 0.]]

# Create an array of ones
ones_array = np.ones((3, 2))  # 3 rows, 2 columns
print(ones_array)
# Output:
# [[1. 1.]
#  [1. 1.]
#  [1. 1.]]

# Create an array with a specific range
range_array = np.arange(0, 10, 2)  # Start, Stop, Step
print(range_array)  # Output: [0 2 4 6 8]

# Create an array with evenly spaced values
linspace_array = np.linspace(0, 1, 5)  # Start, Stop, Number of points
print(linspace_array) # Output: [0.   0.25 0.5  0.75 1.  ]
