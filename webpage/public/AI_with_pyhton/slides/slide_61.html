Data Transformation: Standardization

Standardization is a data scaling technique that transforms data to have a mean of 0 and a standard deviation of 1. This is also known as Z-score normalization.

Why Standardize?

  • Brings features to a similar scale, preventing features with larger values from dominating the model.
  • Improves the performance of algorithms sensitive to feature scaling, such as gradient descent-based algorithms (linear regression, logistic regression, neural networks) and distance-based algorithms (k-nearest neighbors, support vector machines).
  • Helps ensure convergence of algorithms.

Formula

Z = (x - μ) / σ

  • x: The original data point
  • μ: The mean of the feature
  • σ: The standard deviation of the feature

Example

Consider a feature 'Age' with values [20, 30, 40, 50].

Mean (μ) = (20 + 30 + 40 + 50) / 4 = 35

Standard Deviation (σ) ≈ 12.91

Standardized values:

  • (20 - 35) / 12.91 ≈ -1.16
  • (30 - 35) / 12.91 ≈ -0.39
  • (40 - 35) / 12.91 ≈ 0.39
  • (50 - 35) / 12.91 ≈ 1.16

Python Implementation (Scikit-learn)

      
from sklearn.preprocessing import StandardScaler
import numpy as np

data = np.array([[20], [30], [40], [50]])
scaler = StandardScaler()
scaled_data = scaler.fit_transform(data)

print(scaled_data)
