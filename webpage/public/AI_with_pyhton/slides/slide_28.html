Slide 28: Exercise: Dictionary Example - Store and Access Data

Instructions:

In this exercise, you will create and manipulate a Python dictionary to store and access data. Follow the steps below:

  1. Create an empty dictionary called student.
  2. Add the following key-value pairs to the student dictionary:
    • "name": "Alice"
    • "age": 20
    • "major": "Computer Science"
  3. Print the value associated with the key "name".
  4. Update the value associated with the key "age" to 21.
  5. Add a new key-value pair to the student dictionary: "courses": ["AI", "Python", "Data Structures"]
  6. Print the entire student dictionary.

Question 1: What is the data type of the value associated with the key "age" after the update?



Correct Answer: Integer

Question 2: How do you access the first course in the 'courses' list within the dictionary?



Correct Answer: student['courses'][0]
