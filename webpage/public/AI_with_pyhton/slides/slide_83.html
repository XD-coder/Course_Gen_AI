Instructions: Exercise - Impact of C Parameter

Question 1: In the context of Support Vector Machines (SVMs), what does the 'C' parameter control?




Correct Answer: b) The margin of error tolerated by the SVM. A smaller C value allows larger margin violations (more tolerance for errors) while a larger C value penalizes margin violations heavily.

Question 2: If you increase the value of 'C' in an SVM, what is the likely impact on the model?




Correct Answer: b) The model becomes more prone to overfitting. A higher C value encourages the model to fit the training data more closely, potentially leading to overfitting.

Question 3: Which of the following scenarios might warrant using a small value for 'C' in an SVM?




Correct Answer: b) When you want to avoid overfitting and create a more generalized model. A smaller C allows for a larger margin and can improve generalization by allowing some misclassification.
