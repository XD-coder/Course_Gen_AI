Slide 25: Exercise: Even/Odd Checker

Instructions:

Guide the learner to write a program that checks if a number is even or odd.

Consider these questions while building the program:

Quiz Time!

1. What operator is used to find the remainder of a division?


2. What is the condition to check if a number `n` is even in Python?


3. If a number is not even, what is it?


Answers:

1. Modulo operator (%)

2. n % 2 == 0

3. Odd
