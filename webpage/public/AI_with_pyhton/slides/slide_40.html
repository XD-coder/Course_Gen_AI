Slide 40: Matplotlib - Basic Plotting with pyplot

Instructions: Introduce basic plotting using matplotlib.pyplot.

Introduction

Matplotlib is a powerful Python library for creating static, interactive, and animated visualizations in Python. The pyplot module provides a convenient interface to the matplotlib object-oriented plotting library. This slide covers the basics of creating simple plots using matplotlib.pyplot.

Basic Plotting

Here's a simple example of plotting a line graph:


import matplotlib.pyplot as plt

# Sample data
x = [1, 2, 3, 4, 5]
y = [2, 4, 6, 8, 10]

# Create the plot
plt.plot(x, y)

# Add labels and title
plt.xlabel("X-axis")
plt.ylabel("Y-axis")
plt.title("Simple Line Plot")

# Display the plot
plt.show()

Key Functions

  • plt.plot(x, y): Creates a line plot with x and y data.
  • plt.xlabel("label"): Sets the label for the x-axis.
  • plt.ylabel("label"): Sets the label for the y-axis.
  • plt.title("title"): Sets the title of the plot.
  • plt.show(): Displays the plot.

Plot Customization

You can customize plots with various options:

  • Line Styles: Use linestyle (or ls) to change the line style (e.g., '-', '--', ':').
  • Colors: Use color (or c) to change the line color (e.g., 'red', 'green', 'blue').
  • Markers: Use marker to add markers to the data points (e.g., 'o', 'x', '.').

Example:


plt.plot(x, y, linestyle='--', color='red', marker='o')
