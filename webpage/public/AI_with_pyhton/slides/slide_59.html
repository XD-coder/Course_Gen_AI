Data Transformation: Scaling

Data scaling is a crucial preprocessing step in many machine learning algorithms. It involves transforming the range of features to a common scale. This is important because algorithms like gradient descent are sensitive to the scale of the input features. Features with larger values can dominate the learning process.

Common Scaling Methods

1. Min-Max Scaling

Min-Max scaling scales the features to a range between 0 and 1. The formula is:

X_scaled = (X - X_min) / (X_max - X_min)

Example:

from sklearn.preprocessing import MinMaxScaler
import numpy as np

data = np.array([[1, 10], [2, 20], [3, 30]])
scaler = MinMaxScaler()
scaled_data = scaler.fit_transform(data)
print(scaled_data)

2. Standard Scaler (Z-score Normalization)

Standard scaler standardizes features by removing the mean and scaling to unit variance. The formula is:

X_scaled = (X - mean) / standard_deviation

Example:

from sklearn.preprocessing import StandardScaler
import numpy as np

data = np.array([[1, 10], [2, 20], [3, 30]])
scaler = StandardScaler()
scaled_data = scaler.fit_transform(data)
print(scaled_data)
