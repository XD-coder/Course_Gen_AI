Slide 60: Data Transformation - Normalization

What is Data Normalization?

Data normalization is a scaling technique applied during data preparation to change the values of numeric columns in the dataset to a common scale, without distorting differences in the ranges of values. Normalization is used to improve the accuracy of, and speed up the training of machine learning models. It is generally required when different features in a dataset have different ranges.

Common Normalization Methods

1. Min-Max Scaling

Scales and translates each feature individually such that it is in the given range, usually between zero and one.

Formula: X_scaled = (X - X_min) / (X_max - X_min)

Example:

Original data: [10, 20, 30, 40, 50]

Min-Max scaled data (range 0-1): [0, 0.25, 0.5, 0.75, 1]

2. Z-Score Standardization

Transforms the data to have a mean of zero and a standard deviation of one.

Formula: X_scaled = (X - mean) / standard_deviation

Example:

Original data: [1, 2, 3, 4, 5]

Z-Score standardized data (approx.): [-1.41, -0.71, 0, 0.71, 1.41]

3. Robust Scaling

Uses the median and interquartile range (IQR) to scale data, making it more robust to outliers.

Formula: X_scaled = (X - Median) / IQR
