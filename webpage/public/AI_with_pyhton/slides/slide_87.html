Training and Evaluating SVM Models

Instructions: Training and evaluating SVM models in Scikit-learn.

Scikit-learn provides a simple and efficient way to implement SVM models. The sklearn.svm module contains classes for different SVM algorithms, such as SVC for classification and SVR for regression.

Here's a basic example of training and evaluating an SVM classifier:


from sklearn import datasets
from sklearn.model_selection import train_test_split
from sklearn.svm import SVC
from sklearn.metrics import accuracy_score

# Load the iris dataset
iris = datasets.load_iris()
X = iris.data
y = iris.target

# Split the data into training and testing sets
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)

# Create an SVM classifier
svm_model = SVC(kernel='linear', C=1)  # You can adjust the kernel and C parameter

# Train the model
svm_model.fit(X_train, y_train)

# Make predictions on the test set
y_pred = svm_model.predict(X_test)

# Evaluate the model
accuracy = accuracy_score(y_test, y_pred)
print(f"Accuracy: {accuracy}")

Key points:

  • Import necessary modules from sklearn.
  • Load your dataset and split it into training and testing sets using train_test_split.
  • Create an instance of the SVC class, specifying the kernel (e.g., 'linear', 'rbf') and other hyperparameters.
  • Train the model using fit(X_train, y_train).
  • Make predictions on the test set using predict(X_test).
  • Evaluate the model's performance using metrics like accuracy, precision, recall, etc.
