Slide 24: Exercise - Temperature Conversion (Celsius to Fahrenheit)

Instructions:

Write a Python program to convert a temperature from Celsius to Fahrenheit.

Here's a basic structure to get you started:

# Get Celsius temperature from the user celsius = float(input("Enter temperature in Celsius: ")) # Calculate Fahrenheit fahrenheit = (celsius * 9/5) + 32 # Print the result print(f"{celsius} degrees Celsius is equal to {fahrenheit} degrees Fahrenheit")

Remember to:

  • Get user input for the Celsius temperature.
  • Perform the correct conversion calculation.
  • Display the Fahrenheit temperature.

Test Your Understanding:

1. What is the correct formula to convert Celsius to Fahrenheit?





Answer: a) (Celsius * 9/5) + 32

2. If the Celsius temperature is 0, what is the Fahrenheit temperature?





Answer: b) 32
