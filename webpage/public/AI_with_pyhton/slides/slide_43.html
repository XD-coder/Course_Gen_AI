Setting up a Development Environment: Jupyter Notebooks

Instructions:

This slide explains how to install and use Jupyter Notebooks, an essential tool for AI development with Python.

What is Jupyter Notebook?

Jupyter Notebook is an interactive computational environment that lets you combine code, text, images, and visualizations in a single document. It's widely used for data analysis, machine learning, and scientific computing.

Installation:

The recommended way to install Jupyter Notebook is using Anaconda, a Python distribution that includes many useful packages.

  • Anaconda: Download and install Anaconda from the Anaconda website. Choose the Python 3.x version.
  • Using pip: If you prefer using pip, you can install Jupyter Notebook with the following command:
    • pip install jupyter

Running Jupyter Notebook:

After installation, you can start Jupyter Notebook from the command line:

jupyter notebook

This will open a new tab in your web browser with the Jupyter Notebook interface.

Creating a New Notebook:

In the Jupyter Notebook interface:

  • Click the "New" button.
  • Select "Python 3" (or another kernel if you have installed others).

This will create a new notebook file (.ipynb) where you can write and execute Python code.

Basic Usage:

  • Cells: Jupyter Notebooks are organized into cells. You can have code cells (for writing Python code) and Markdown cells (for writing text, headings, etc.).
  • Executing Cells: To execute a cell, press Shift + Enter. The output will be displayed below the cell.
  • Markdown: You can write formatted text in Markdown cells.
  • Saving: Jupyter Notebooks are automatically saved, but you can also save manually using the "File" menu.
