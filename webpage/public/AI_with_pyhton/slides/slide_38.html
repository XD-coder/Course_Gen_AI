Slide 38: Pandas - Descriptive Statistics

Instructions: Explain how to calculate descriptive statistics using Pandas

Pandas provides powerful functions for calculating descriptive statistics on DataFrames and Series. Here's a breakdown:

Key Functions:

  • .describe(): Generates a summary of descriptive statistics including mean, standard deviation, min, max, quartiles (25%, 50%, 75%) for numerical columns.
  • .mean(): Calculates the mean (average) of a column or Series.
  • .median(): Calculates the median (middle value) of a column or Series.
  • .std(): Calculates the standard deviation of a column or Series.
  • .min(): Finds the minimum value in a column or Series.
  • .max(): Finds the maximum value in a column or Series.
  • .count(): Counts the number of non-null values in a column or Series.
  • .value_counts(): Counts the frequency of each unique value in a Series.
  • .corr(): Calculates the correlation matrix between numerical columns in a DataFrame.

Example:


  import pandas as pd

  # Sample DataFrame
  data = {'col1': [1, 2, 3, 4, 5],
          'col2': [6, 7, 8, 9, 10]}
  df = pd.DataFrame(data)

  # Descriptive Statistics
  print(df.describe())

  # Mean of col1
  print("Mean of col1:", df['col1'].mean())

  # Median of col2
  print("Median of col2:", df['col2'].median())

.describe() provides a quick overview. Individual functions let you calculate specific statistics as needed.
