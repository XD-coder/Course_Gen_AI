Slide 18: Data Structures: Sets

Instructions: Explain sets, uniqueness, and set operations

What is a Set?

A set is an unordered collection of unique elements. This means that no element can appear more than once in a set.

Uniqueness

The defining characteristic of a set is that it only contains unique values. If you try to add a duplicate element, it will simply be ignored.

Set Operations

Sets support various operations:

  • Union: Combines elements from two sets.
  • Intersection: Returns elements present in both sets.
  • Difference: Returns elements present in the first set but not in the second.
  • Subset/Superset: Checks if one set is contained within another.
