Installing Python and pip on Linux

Instructions: Explain how to install Python on Linux using the system's package manager (e.g., apt, yum).

On most Linux distributions, you can install Python using the system's package manager. This is the recommended method as it ensures that Python is properly integrated with your system and that dependencies are managed correctly.

Using apt (Debian/Ubuntu):

sudo apt update
sudo apt install python3 python3-pip

Using yum (CentOS/RHEL/Fedora):

sudo yum update
sudo yum install python3 python3-pip

Using dnf (Fedora 22+):

sudo dnf update
sudo dnf install python3 python3-pip

After installation, verify the installation by checking the Python and pip versions:

python3 --version
pip3 --version
