Slide 31: Using pip

Installing Packages

To install a package using pip, use the following command in your terminal:

pip install package_name

Replace package_name with the name of the package you want to install (e.g., pip install numpy).

Upgrading Packages

To upgrade a package to the latest version, use the following command:

pip install --upgrade package_name

This will update the specified package to its newest available release.

Uninstalling Packages

To uninstall a package, use the following command:

pip uninstall package_name

This will remove the package from your Python environment.

Additional Notes:

  • Use pip list to see all installed packages.
  • Use pip show package_name to get information about a specific package.
  • Consider using virtual environments to manage dependencies for different projects.
