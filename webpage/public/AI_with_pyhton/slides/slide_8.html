Python Data Types: int, float, str, bool

int (Integer)

Represents whole numbers (positive or negative) without any decimal points.

Example: x = 10, y = -5, z = 0

float (Floating-Point Number)

Represents numbers with decimal points.

Example: pi = 3.14, price = 99.99, temp = -27.5

str (String)

Represents a sequence of characters enclosed in single quotes ('...') or double quotes ("...").

Example: name = "Alice", message = 'Hello, world!'

bool (Boolean)

Represents either True or False. Used for logical operations.

Example: is_valid = True, is_finished = False
