Slide 70: Cost Function - Mean Squared Error (MSE)

Instructions: Explain the concept of a cost function - MSE

In machine learning, a cost function, also known as a loss function, is a measure of how well a machine learning model is performing. It quantifies the difference between the predicted values generated by the model and the actual (true) values in the dataset. The goal of training a machine learning model is to minimize this cost function, effectively improving the model's accuracy and predictive power.

The Mean Squared Error (MSE) is a common cost function, particularly used in regression problems. It calculates the average of the squared differences between the predicted and actual values. A lower MSE indicates a better fit, as it signifies smaller average errors.

Mathematically, MSE is calculated as:

MSE = (1/n) * Σ(yi - ŷi)^2

where:

  • n is the number of data points
  • yi is the actual value for the i-th data point
  • ŷi is the predicted value for the i-th data point
