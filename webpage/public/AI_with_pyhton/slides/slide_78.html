Slide 78: Introduction to Logistic Regression

Instructions:

Explain the goal of logistic regression: to predict the probability of a binary outcome.

Question 1: What type of outcome does logistic regression primarily predict?



Correct Answer: Categorical (binary) values

Question 2: Logistic regression uses which function to map inputs to probabilities?



Correct Answer: Sigmoid Function

Question 3: True or False: Logistic regression can be used for multi-class classification without modification.


Correct Answer: False
